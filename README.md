# Rental Sale Calculator

A sleek, multi-step PWA that compares cash flow, cap rate, and cash-on-cash return between your current rental and a potential new property.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## PWA

The app ships with a web manifest and service worker (via `next-pwa`). The service worker is disabled in development and enabled in production builds.
